# Přílohy (obrázky, PDF, binárky)

> **05-RESOURCES** = kurátorované reference (profil, inspirace). **Přílohy** sem patří jen jako úložiště souborů — ne jako „dokumenty k čtení“.

## Kam co dát

| Situace | Složka | Příklad |
|---------|--------|---------|
| Nový paste ve vaultu (Obsidian default) | `_paste/` | Dočasně, pak přesuň |
| K archivovanému INBOX záznamu | `07-ARCHIV/inbox-processed/…/` vedle `.md` | Screenshot u meetupu |
| K výstupu projektu | `02-PROJEKTY/<slug>/` (složka hubu) | Wireframe, export |
| Cross-projekt reference | `05-RESOURCES/<téma>/` | Profil, šablony |

## Po zpracování capture

1. Přesuň soubor z `_paste/` nebo z rootu vaultu.
2. Pojmenuj popisně: `YYYY-MM-DD-<kontext>.jpg`.
3. Uprav wikilink v poznámkách (`alwaysUpdateLinks` v Obsidianu to často opraví samo).

## Co nedělat

- Nechat `Pasted image …` v **rootu** vaultu.
- Dávat vše do `05-RESOURCES/` bez podsložky — zamotá to s `profil-rb-edu/`, `inspirace/`.

_Viz [[Jak čtu vault MrLUC]] · Obsidian: Settings → Files → Default location for new attachments = `05-RESOURCES/attachments/_paste`_
